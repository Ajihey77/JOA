## 공부 방법

1. 공부할 거리를 찾아옵니다.
2. 기간을 정합니다. (기간은 다같이 정합니다)
3. 기간에 맞춰 공부합니다. (시간: 평일 점심시간)
4. 기간안에 공부하지 못한 인원은 탈락합니다.(회생을 원할 시 점심을 한번 사야합니다.)

## 공부 과정

1. 한달에 한번 공부한 내용을 앞에서 발표 및 공유 합니다.
2. 발표를 듣고 서로 피드백을 주고 받습니다.
